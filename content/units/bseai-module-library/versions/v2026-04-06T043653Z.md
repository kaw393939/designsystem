  - type: section
    id: using-the-library
    title: A module is not the same thing as a route or a week.
    body: >
      Public routes keep the site legible. Canonical modules keep the teaching
      reusable. Course wrappers add dates, deadlines, critique rhythm, and
      submission rules.


      The module library exists so the degree can grow without forcing every
      term or every route to reinvent the same learning block. It also lets the
      site stay small at the top while still holding deeper instructional logic
      underneath.


      Use the library when the learner needs a durable explanation, a reusable
      comparison surface, or a stable artifact model. Use the wrapper when the
      learner needs timing, grading, or this-week direction.

# BSEAI Content Production Specs

## Purpose

This folder is the execution-layer spec set for producing BSEAI content up to the point where frontend implementation begins.

It exists to keep the content team, QA flow, and autonomous authoring process aligned with the split IA and the file-backed publishing workflow.

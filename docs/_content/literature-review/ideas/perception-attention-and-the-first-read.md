# Perception, Attention, and the First Read

## Why this topic matters here

The project repeatedly argues that the page is judged before it is fully read.

That means perception is not secondary. It is part of meaning.

## What the internal research argues

The internal documents make six strong claims:

1. perception shapes what the page feels like in the first second
2. Swiss-style order tends to earn trust
3. disruptive or brutalist moves can interrupt the scroll
4. faces and gaze can buy attention, but only if they support meaning
5. decorative visuals do not count as strategy
6. style is a signaling system, not ornament

## What the literature supports

### Visual attention is selective and structured

Treisman and Gelade, Posner, Itti, Koch, and Niebur, and later visual-search work all support the basic claim that attention is not neutral. It is guided by features, expectations, goals, and salience.

This matters because page design is always shaping what gets noticed first.

### First impressions form very quickly

Lindgaard and colleagues’ paper is now a standard reference because it compresses the challenge clearly: first impressions of web pages form almost immediately.

The title itself is the warning:

> “You have 50 milliseconds to make a good first impression!”

Even if the exact timing varies across settings, the usable lesson stands: the first read is fast, partial, and decisive.

### Faces and gaze cueing matter, but not magically

Friesen and Kingstone, Driver and colleagues, and later gaze-cueing reviews support the claim that eyes and gaze direction can orient attention.

This helps justify the project’s interest in portrait use and gaze direction, but it also adds a guardrail: the face is not the message. It is an attention tool that can support the message if used deliberately.

### Banner blindness matters for portfolio design too

Benway and Lane’s banner blindness work is important because it reminds the project that salience alone does not produce attention. If an element looks like an ad, users may ignore it even when it is prominent.

This is one reason why credibility, hierarchy, and visual fit matter as much as interruption.

### Grid logic is part of trust production

Müller-Brockmann’s grid tradition and Frost’s design-systems work matter because they give the project a formal vocabulary for order, repeatability, and structural coherence.

The project’s core insight is that layout is not only aesthetic. It is pedagogical and rhetorical.

## What this means for the project

### For the route experience

1. every major page needs a strong first-read contract
2. style choice should follow the signal and page job
3. proof should not be buried below ornamental hero space
4. faces, diagrams, and imagery should be used as attention anchors, not filler

### For future testing

1. test face versus no-face hero variants
2. test gaze direction toward headline or CTA
3. test calm grid-first versions against louder interrupt-first versions
4. track headline comprehension, CTA discovery, and proof visibility

## Cautions

1. do not turn a face heuristic into a fake universal rule
2. do not assume disruption always improves outcomes
3. do not use Swiss order as a synonym for lifelessness
4. do not use visual tension when the page really needs calm trust

## Working doctrine

The first read is a design problem and a cognition problem at the same time. The page should interrupt selectively, clarify quickly, and hand off to proof before attention collapses.

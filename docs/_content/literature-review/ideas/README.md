# Ideas Index

## Purpose

This folder holds the topic papers that turn the broader research stack into a usable conceptual review.

Each file isolates one major thread in the project thesis so maintainers can read deeply on a single topic without losing the larger frame.

## Current papers

- `identity-archetypes-and-coherent-self-presentation.md`: identity coherence, archetypes, and legible self-presentation
- `motivation-needs-and-self-actualization.md`: human need, development, and motivational structure
- `perception-attention-and-the-first-read.md`: first-read cognition, attention, and interpretive hierarchy
- `trust-persuasion-and-ethical-influence.md`: trust, proof, and non-exploitative persuasion
- `signaling-weak-ties-and-opportunity.md`: signaling, network effects, and opportunity access
- `virtue-self-command-and-whole-person-formation.md`: character, self-command, and whole-person education
- `renaissance-reformation-primary-texts-and-the-ai-authority-shift.md`: print-era comparison material for authority and knowledge shifts
- `second-renaissance-ai-and-transformative-education.md`: AI-era institutional change and the broader educational thesis

Treat these papers as synthesis outputs, not as raw sources.

When you need the evidence backbone beneath a claim in one of these files, step back to `../references/`, `../people/`, or `docs/_research/`.

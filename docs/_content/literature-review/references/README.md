# References Index

## Purpose

This folder is the evidence spine for the literature-review package.

It keeps the source backbone compact so topic papers can stay readable while maintainers still have a clear trail for fact-checking, charting, and citation work.

## Current files

- `annotated-bibliography.md`: working reading list with short notes on relevance and use
- `data-source-register.md`: source register for datasets, public evidence bases, and chart-ready material

## How to use this folder

Use `annotated-bibliography.md` when you need to understand why a source belongs in the corpus.

Use `data-source-register.md` when you need to verify a number, plan a chart, or identify which external source underpins a claim about labor, education, governance, or AI.

## Boundary rule

Do not turn this folder into a second copy of the topic papers.

Keep interpretation in `../ideas/` and keep this folder focused on provenance, evidence planning, and source control.

# Quote Bank

## Purpose

This folder stores short, verified quotations that are reusable across:

1. teaching slides
2. editorial copy
3. route headers and callouts
4. image and illustration direction

Only brief quotations belong here.

Long excerpts should stay in notes, not in the reusable quote bank.

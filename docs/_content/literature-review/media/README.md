# Media Planning

## Purpose

This folder is the visual-planning layer for the research corpus.

It covers three kinds of assets:

1. image prompts for illustrations, people, and branding
2. chart plans tied to real data sources
3. generated concept assets that can inform future website art direction
4. audio production notes for narrated explainers and long-form companions

## Usage rules

1. Generated images here are concept assets, not final production art.
2. Real-person historical portraits should use licensed or public-domain source material unless a clearly synthetic treatment is intended.
3. Data charts should be redrawn from cited numbers rather than invented from image prompts.

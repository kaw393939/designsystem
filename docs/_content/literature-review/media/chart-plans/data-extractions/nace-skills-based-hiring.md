# NACE: Skills-Based Hiring

## Source

National Association of Colleges and Employers (NACE), Job Outlook reports 2024-2026.

Source URL: <https://www.naceweb.org/job-market/trends-and-predictions/almost-two-thirds-of-employers-use-skills-based-hiring-to-help-identify-job-candidates>

Retrieval date: 2026-04-05

## Key findings

| Metric | Value | Survey period |
| --- | --- | --- |
| Employers using skills-based hiring for entry-level (2025 report) | 64.8% | Aug-Sep 2024 |
| Employers using always or most of the time | more than 50% | Aug-Sep 2024 |
| GPA used as a hiring screen (2024 report) | less than 40% | 2023 survey |
| Projected skills-based hiring (2026 cycle) | nearly 70% | Aug-Sep 2025 |

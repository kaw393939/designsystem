# Generated Media Index

## Purpose

This folder contains generated concept assets used for art direction, visual exploration, and future production planning.

These files are not final shipped assets. They are working visual references tied to the literature-review package.

## Current sets

- `archetypes/`: six archetype concept images for explorer, hero, innocent, magician, outlaw, and sage
- `branding/`: identity-system and second-renaissance poster-style concept boards
- `illustrations/`: system diagrams and editorial scenes such as trust architecture, first-read hierarchy, and weak-ties networks
- `people/`: portrait-style scenes for mentor, studio, and student-builder imagery

## Use rule

Use these files to guide composition, mood, and visual language decisions.

If an asset becomes part of a production route or release, move or recreate it in the appropriate published asset path rather than treating this folder as the final source of truth.

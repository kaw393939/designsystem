# Literature Review Package

## Purpose

This folder turns the existing research stack into a working literature-review package.

It does three jobs:

1. integrates the ideas already developed in `docs/_research/`
2. adds externally researched source trails with citations and URLs
3. organizes the material into expandable folders for ideas, luminaries, and institutional evidence

## Folder structure

- `00-synthesis/`: integrated outline, source map, and the master framing for the project
- `ideas/`: topic papers that translate the research stack into a usable conceptual review
- `luminaries/`: grouped overview notes on the key thinkers and operators behind each cluster
- `people/`: one file per scholar, operator, or major practitioner for deeper citation work
- `quotes/`: verified short quotations organized for teaching, writing, and art direction
- `references/`: annotated bibliography and data-source registers for evidence planning
- `institutions/`: reports, standards, and evidence bases that ground claims about AI, labor, governance, and student readiness
- `media/`: image briefs, chart plans, and generated concept assets for future sites and teaching media

## Method

This package was built from two source layers:

1. Internal synthesis from:
   - `docs/_research/identity-system-core.md`
   - `docs/_research/identity-system-maintainer-doctrine.md`
   - `docs/_research/identity-system-student-handout.md`
   - `docs/_research/identity.md`
   - `docs/_research/mysystem.md`
   - `docs/_research/renesaince.md`
2. External research discovery and source retrieval using the local `eai` CLI plus direct source verification where needed

## How to use this package

Start here:

1. `00-synthesis/integrated-research-outline.md`
2. `00-synthesis/source-map.md`
3. `00-synthesis/bookshelf-to-content-map.md`

Then move through the topic papers in `ideas/`.

Use `luminaries/` when you need the grouped overview for a research cluster.

Use `people/` when you need one scholar at a time for citations, quotes, sourcing, or future profile pages.

Use `quotes/` when you need short, reusable lines for teaching slides, page copy, or editorial framing.

Use `references/` when you need the source backbone for annotated reading lists, chart production, or fact-checking.

Use `institutions/` when you need hard evidence, standards, labor-market framing, or public-health/education baselines.

Use `media/` when you need image prompts, chart storyboards, or generated concept assets.

## Current research thesis

The core project is not only a portfolio system.

It is a broader framework for public legibility in an AI-shaped world.

That framework links:

1. motivation and human need
2. identity formation and archetypal coherence
3. perception, hierarchy, and first-read cognition
4. trust, proof, and ethical persuasion
5. signaling, weak ties, and opportunity formation
6. AI-era labor shifts and the need for public evidence of capability
7. whole-person, transformation-oriented education in a possible “second renaissance” moment

## Current expansion layers

The new corpus layers add:

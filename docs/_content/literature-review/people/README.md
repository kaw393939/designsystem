# People Library

## Purpose

This folder turns the grouped luminary overviews into a one-file-per-person research library.

Use these files when you need:

1. a tighter citation trail for one thinker
2. a cleaner source list for writing or teaching
3. a future profile page, quote card, or scholar explainer

The grouped overview files still live in `../luminaries/`.

This folder is the deep archive beneath those overviews.

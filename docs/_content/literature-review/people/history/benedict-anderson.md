# Benedict Anderson

## Why he matters here

Anderson explains how media systems help produce large-scale imagined communities and shared public discourse.

He matters because the project is not only about jobs or portfolios. It is also about public legibility and social imagination.

## What this project should take

1. Media systems shape collective identity.
2. Public discourse depends on shared symbolic environments.
3. Communication forms can organize social belonging.

## What to handle carefully

1. Do not use imagined communities as a vague metaphor for every audience.
2. Keep Anderson connected to actual media and institutional analysis.

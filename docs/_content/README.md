# Content Research Index

## Purpose

This folder holds synthesized research outputs that sit between the raw research layer and live site content.

It is where source material becomes:

1. literature-review papers
2. integrated outlines
3. course-facing support documents
4. media and citation planning artifacts

## Current structure

- `literature-review/`: the main synthesis package, including idea papers, source maps, references, quotes, people notes, and media planning
- `identity-portfolio-course-outline.md`: course-shape material derived from the identity and opportunity system

## Relationship to other folders

- use `docs/_research/` for source texts, internal doctrines, and primary inputs
- use `docs/_content/` for derived argument and curriculum-facing synthesis
- use `content/` at the repo root for file-backed units, experiences, releases, and the operational publishing library

## Reading order

Start here:

1. `literature-review/README.md`
2. `literature-review/00-synthesis/integrated-research-outline.md`
3. `literature-review/00-synthesis/source-map.md`
4. `literature-review/00-synthesis/bookshelf-to-content-map.md`

## Maintenance rule

If a new synthesis package is added beside `literature-review/`, add it to this index and state how it differs from the source layer in `docs/_research/`.

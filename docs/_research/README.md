# Research Index

## Purpose

This folder holds the source research layer for the project.

These files are the internal foundation beneath the site, the literature-review package, and the broader educational thesis.

## Structure

- `books/`: audiobooks, public-domain classics, transcript artifacts, and the working bookshelf guide
- `identity-system-core.md`: primary student-facing model for the identity portfolio system
- `identity-system-student-handout.md`: shortest teaching version of the identity model
- `identity-system-maintainer-doctrine.md`: maintainer rules for keeping the identity experience coherent
- `identity.md`: extended evidence base and conceptual validation for the identity system
- `mysystem.md`: compressed operator blueprint for the identity system
- `renesaince.md`: broader institutional and educational framing for the second-renaissance argument

## Reading order

When working on identity and portfolio content, read in this order:

1. `identity-system-core.md`
2. `identity-system-student-handout.md`
3. `identity-system-maintainer-doctrine.md`
4. `identity.md`
5. `mysystem.md`

When working on institutional, AI-era, or educational arguments, read:

1. `renesaince.md`
2. `books/README.md`
3. the relevant classics or transcript artifacts under `books/`

## Boundary rule

This folder is for source material and durable internal models.

If a document is mainly synthesizing sources into publishable argument, route planning, or literature-review outputs, it belongs under `docs/_content/` instead.

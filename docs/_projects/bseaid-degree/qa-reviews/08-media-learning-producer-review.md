# Media Learning Producer Review

## Status

- Lens: media learning producer
- Review target: IA media package rules and site-level production plan

## Findings

1. The media package rule is solid, but it still needs a reusable script and asset package template for room-level production.
2. The route-by-route media plan identifies likely audio needs, but not a decision rule for what earns long-form audio versus short narration versus no audio.
3. The production workflow covers generation and review, but it lacks a formal QA gate for narration, pacing, and companion-note coherence.

## Risks

1. Media production may start before scripts and note maps are normalized.
2. Audio may proliferate unevenly across routes.
3. Long-form assets may sound like spoken page copy rather than intentional teaching media.

## Required changes

1. Add a script package template including title, audience, learning job, script, chapter map, paired visual, and transcript note.
2. Add an audio selection rubric so not every room automatically gets long-form narration.
3. Add a media QA checkpoint requiring script review, pronunciation review, pacing review, and visual pairing review.

# docs2/research — README

This folder holds the research corpus organized by degree module.

Populated on demand when a specific module requires citations, quotes, institutional evidence, or media planning.

## Principle

Do not populate this folder speculatively. Generate module research packs when a module is actively being authored or taught.

Each module folder should contain:

- `claims.md` — the working claims that have strong enough support to use in teaching and public writing
- `citations.md` — the key sources organized by claim, with retrieval notes
- `quotes.md` — extractable quotations with attribution and context
- `evidence.md` — institutional data, charts, and statistics with source provenance
- `open-questions.md` — what the research does not yet settle

## Module folders (created when needed)

- `why-now/` — AI as institutional shift, economic history, labor market evidence
- `identity-and-signal/` — identity formation, archetype systems, cohesion science
- `visual-design-literacy/` — design lineages, attention research, hierarchy and trust
- `trust-and-persuasion/` — persuasion science, ethical influence, behavioral economics
- `proof-and-deployment/` — signaling theory, weak ties, job access, public narrative
- `formation-model/` — the updated trivium/quadrivium, classical formation philosophy, AI-era implications
- `agentic-systems/` — LLM orchestration, enterprise AI infrastructure, evaluation and governance
